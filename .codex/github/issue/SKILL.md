# issue

Purpose
-------
Create the tracked work item for one bounded workflow slice.

Procedure
---------
1. Identify the bounded slice of work.
2. Ground it in the controlling source-of-truth artifact or concrete review
   finding.
3. Draft:

   - problem
   - scope
   - non-goals
   - acceptance criteria
   - validation / conformance checks
   - references

4. Provide or run the `gh issue create` command.
5. Recommend `branch` as the next skill once the issue exists.
