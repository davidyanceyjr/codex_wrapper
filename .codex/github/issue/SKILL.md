# issue

Purpose
-------
Create the tracked work item for one bounded workflow slice.

Procedure
---------
1. Identify the bounded slice of work.
2. Ground it in the controlling source-of-truth artifact or concrete review
   finding.
3. Draft:

   - problem
   - scope
   - non-goals
   - acceptance criteria
   - validation / conformance checks
   - references

4. Present the exact issue title/body for approval.
5. After explicit approval, create the issue with `gh issue create` or show the
   exact command that will be run.
6. Recommend `branch` as the next skill once the issue exists.
